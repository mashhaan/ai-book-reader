## Page 1

Лабораторна робота 4
Тема: Створення базової HTML/CSS розмітки

Мета роботи. Набуття практичних навичок створення веб-сторінок простої структури з використанням базових можливостей HTML та CSS.

Завдання
1. Продумати структуру майбутнього проекту (за типом файлів).
   Підготувати необхідні теки.
2. Проаналізувати дизайн та структуру вебсайту, який Ви створили у Figma на попередній лабораторній роботі. Експортувати усі необхідні зображення з Figma до відповідних тек проекту.
3. Створити файл index.html.
4. Створити HTML-структуру згідно з макетом, використовуючи теги HTML для створення заголовків, текстових блоків, списків та інших елементів.
5. Обов'язково використовувати семантичні теги.
6. Додати необхідні CSS-стилі для розміщення елементів на сторінці відповідно до макету. Використовуйте CSS-селектори та властивості для задання розмірів, кольорів, фонів, шрифтів, відступів тощо.
7. Перевіріть сторінки на відповідність стандартам HTML та CSS за допомогою валідаторів.

Вимоги до звіту
1. Проект має розміщуватися у GIT (у Tims кріпите лише посилання на проект).
2. Звіт про пророблену роботу виконуєте у вигляді відеозапису, у якому демонструєте отримані результати та коментуєте їх.
